# GDB

Compile with debugging enabled:

    gcc -g hello.c

Start GDB

    gdb a.out

### Commands:

Break main

    b main

Run

    r

Step

    s

Print variables:
    
    p [variable]
    info locals

List code

    l

Quit 

    q
